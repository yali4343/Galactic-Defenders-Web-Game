About Galactic Defenders



Game Instructions:

  • Use arrow keys to move your spaceship
  • Press the configured key to shoot at the spaceships
  • Different rows of enemy spaceships give different points when destroyed
  • Avoid getting hit by the enemy spaceships' rockets
  • Destroy all ememy spaceships before time runs out

Challenges Faced:

We faced several challenges while developing this game:

  • Implementing the game logic and ensuring smooth gameplay
  • Creating a responsive design that works on different screen sizes
  • Handling user input and managing game state effectively
  • Debugging and testing the game to ensure a bug-free experience
  • Integrating audio and visual effects to enhance the gaming experience
